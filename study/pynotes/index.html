print "Hello, " + x;

    第2章：列表和元组

    2.1 序列概览

    Python 6种内建序列：

    • 列表；
    • 元组；
    • 字符串
    • Unicode字符串
    • buffer对象
    • xrange对象

    列表可以修改，元组不可修改。

    2.2 通用序列操作

    2.2.1 索引

    >>> greeting = "Hello";
>>> greeting[0];
'H'
>>> greeting[-1];
'o'

    2.2.2 分片

    >>> numbers = "0123456789";
>>> numbers[3:6];
'345'
>>> numbers[7:-1];
'78'
>>> numbers[7:];
>'789'

    加入步长：

    >>> numbers[0:10:2];
'02468'
>>> numbers[10:0:-2];
'97531'

    2.2.3 序列相加

    >>> [1, 2, 3] + [4, 5, 6];
[1, 2, 3, 4, 5, 6]

    2.2.4 乘法

    >>> 'abc' * 3;
'abcabcabc'
